# SigMamba: Missing-Aware Time Series Modeling with Signature and Mamba

## 📌 Overview

This repository contains the official implementation of **SigMamba**, a novel framework for multivariate time series modeling under missing data.

---

---

## ⚙️ Installation

We recommend using Conda:

```bash
conda create -n sigmamba python=3.10
conda activate sigmamba
pip install -r requirements.txt
```

---

## 🚀 Usage


```bash
python train_missing_sigmamba.py --dataset --missing_rate
```
